# React Vercel Supabase

Contoh project React + Vite yang terhubung ke Supabase dan siap dideploy ke Vercel.

## Tutorial

Panduan lengkap setup dan bahan yang dibutuhkan ada di file berikut:

- [tutorial_react_vercel_supabase.md](./tutorial_react_vercel_supabase.md)

## Menjalankan Project

1. Salin `.env.example` menjadi `.env.local` atau `.env`
2. Isi `VITE_SUPABASE_URL` dan `VITE_SUPABASE_PUBLISHABLE_KEY`
3. Jalankan SQL pada `supabase/todos_schema.sql`
4. Jalankan `npm install`
5. Jalankan `npm run dev`
